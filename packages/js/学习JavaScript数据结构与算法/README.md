# 学习JavaScript数据结构与算法

## 作者

[巴西] 格罗纳（Loiane Groner）

## 导读

本书**首先**介绍了`JavaScript`语言的基础知识.

**接下来**讨论了`数组`、`栈`、`队列`、`链表`、`集合`、`字典`、`散列表`、`树`、`图`等数据结构.

**之后**探讨了各种`排序`和`搜索`算法, 包括

- 冒泡排序
- 选择排序
- 插入排序
- 归并排序
- 快速排序
- 顺序搜索
- 二分搜索

还介绍了`动态规划`和`贪心`算法等常用的高级算法及相关知识。
