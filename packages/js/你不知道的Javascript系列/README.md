# 你不知道的Javascript系列 - (共3卷)

## 作者

[美] Kyle Simpson

## 导读

`JavaScript`语言有很多复杂的概念, 但却用简单的方式体现出来（比如`回调函数`）.

`你不知道的JavaScript`系列就是要让不求甚解的`JavaScript`开发者迎难而上, 深入语言内部, 弄清楚`JavaScript`每一个零部件的用途.

本书介绍了该系列的两个主题:

- 作用域和闭包
- this和对象原型

掌握了这些知识之后, 无论什么技术、框架和流行词语, 你都能轻松理解.
