# JavaScript函数式编程

## 作者

[美] Michael Fogus

## 导读

`JavaScript`是近年来非常受瞩目的一门编程语言, 它既支持`面向对象`编程, 也支持`函数式`编程.

本书专门介绍`JavaScript`函数式编程的特性.

全书共`9`章, 分别介绍了:

- `JavaScript`函数式编程
- 一等函数与`Applicative`编程
- 变量的作用域和闭包
- 高阶函数
- 由函数构建函数
- 递归
- 纯度和不变性
- 更改政策
- 基于流的编程
- 类编程

除此之外, 附录中还介绍了更多`函数式JavaScript`.

本书内容全面, 示例丰富, 适合想要了解函数式编程的J`avaScript程序员`和学习JavaScript的`函数式程序员`阅读.
