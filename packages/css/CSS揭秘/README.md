# CSS揭秘

## 作者

[希] Lea Verou

## 导读

本书是一本注重实践的教程, 作者为我们揭示了`47`个鲜为人知的`CSS`技巧.

主要内容包括:

- 背景与边框
- 形状
- 视觉效果
- 字体排印
- 用户体验
- 结构与布局
- 过渡与动画等...

本书将带领读者循序渐进地探寻更优雅的解决方案, 攻克每天都会遇到的各种网页样式难题.

本书的读者对象为`前端工程师`、`网页开发人员`.
